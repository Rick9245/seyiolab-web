Configuration Resend pour le formulaire

Dans Vercel > Project Settings > Environment Variables, ajoute :
RESEND_API_KEY = ta clé API Resend
CONTACT_FROM_EMAIL = Seyio Lab <contact@seyiolab.com>

Dans Resend, vérifie le domaine seyiolab.com puis ajoute les DNS demandés chez IONOS.
Le formulaire envoie vers contact@seyiolab.com.
